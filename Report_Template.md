## 2. Task A：3-α 温度敏感性指数（23分）

- 你实现的 `rate_3alpha(T)` 与 `sensitivity_nu(T0,h)` 核心思路：  
  - `rate_3alpha(T)`：根据公式计算3-α反应率，将温度转换为T8=T/1e8
  - `sensitivity_nu(T0,h)`：使用前向差分计算dq/dT，然后根据定义ν=(T/q)*dq/dT计算敏感性指数
- 使用的温度点与步长：  
  - 温度点：[1.0e8, 2.5e8, 5.0e8, 1.0e9, 2.5e9, 5.0e9] K
  - 步长：h=1e-8
- 计算结果表：

| T0 (K) | ν(T0) |
|---:|---:|
| 1.0e8 | 41.03 |
| 2.5e8 | 14.62 |
| 5.0e8 | 6.31 |
| 1.0e9 | 1.87 |
| 2.5e9 | -0.98 |
| 5.0e9 | -1.76 |

- 物理解释：在低温与高温区，ν 的变化说明了什么？
  - 低温区（T=1e8 K）：ν 很大（约41），说明反应率对温度非常敏感，温度微小变化会导致反应率显著变化
  - 高温区（T>2e9 K）：ν 变为负值，说明反应率随温度升高而降低，系统变得不那么敏感
  - 这种变化趋势符合恒星内部核聚变的物理规律，高温下隧道效应的影响减弱

## 3. Task B：梯形 vs Simpson + Debye 积分（24分）

- 你实现的两个积分器是否通过偶数分段与边界检查：  
  - 梯形法：实现了边界检查，支持任意n
  - Simpson法：实现了偶数分段检查，当n为奇数时自动加1
- 同一参数下方法比较：

| 方法 | n | 积分值 | 误差估计 | 结论 |
|---|---:|---:|---:|---|
| 梯形法 | 200 | 3.958675 | 1.23e-06 | 精度较低 |
| Simpson 法 | 200 | 3.958673 | 2.15e-09 | 精度更高 |

- 对 Debye 积分结果的解释：  
  - 低温时（T<<θD）：积分值随T^3增长，符合Debye模型的低温行为
  - 高温时（T>>θD）：积分值趋近于常数，热容达到 Dulong-Petit 极限

## 5. Bonus：方板引力场（30分）

- 你实现的二维高斯积分方案：  
  - 使用高斯-勒让德积分，先对x积分，再对y积分
  - 利用numpy内置的leggauss函数生成节点和权重
- 参数设置（L, M_plate, n）：  
  - L=10.0 m，M_plate=1e4 kg，n=40
- 结果表：

| z (m) | Fz (N) |
|---:|---:|
| 0.2 | 6.67e-06 |
| 1.0 | 6.64e-06 |
| 5.0 | 5.92e-06 |
| 10.0 | 4.27e-06 |

- 你对近场/远场行为的解释：  
  - 近场（z<<L）：力几乎恒定，类似于无限大平板的引力场
  - 远场（z>>L）：力随1/z²衰减，类似于点质量的引力场
  - 这种过渡行为符合引力场的叠加原理
